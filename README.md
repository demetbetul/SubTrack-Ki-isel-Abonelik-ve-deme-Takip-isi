# SubTrack-Kişisel Abonelik ve Ödeme Takipçisi
SubTrack, dijital ve fiziksel aboneliklerin (Netflix, Spotify, spor salonu, faturalar vb.) merkezi bir noktadan yönetilmesini sağlayan, bütçe dostu bir finansal asistan uygulamasıdır. Dağınık haldeki ödeme döngülerini mühendislik disipliniyle analiz ederek kullanıcıya sunar.
